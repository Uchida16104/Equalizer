# Equalizer

Equalizer is a cross-platform, browser-based audio equalizer designed for:
- People with auditory hypersensitivity
- People with low voice volume
- Real-time frequency-based audio control needs

## Features

- Live microphone input
- FFT-based frequency analysis
- Adjustable target frequency
- Gain and volume control
- No backend processing required
- Works on desktop and mobile browsers

## Technology Stack

- HTMX (latest)
- Tailwind CSS (latest)
- Alpine.js (latest)
- Web Audio API
